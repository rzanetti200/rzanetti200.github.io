Wikipedia-Web-Scraper

This script extracts text content and image links from a given Wikipedia page, counts the frequency of words within the text, and saves the results to a CSV file. It makes an HTTP request to fetch the Wikipedia page's HTML, and utilizes BeautifulSoup to parse and extract paragraphs and image tags. The word frequencies are calculated by tokenizing the text and counting occurrences of each word, after removing non-alphanumeric characters. Finally, the script writes the total word count, the top N word frequencies, and the image links to an output CSV file.
